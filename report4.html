Answers to Questions

What is selector’s specificity and how it’s calculated?

The selector's specificity determines which CSS rule takes precedence when multiple rules target the same element. It's calculated based on the types of selectors used, where ID selectors have the highest specificity, followed by class selectors, attribute selectors, and element selectors. Inline styles have the highest specificity and override other styles.

What are complex selectors?

Complex selectors are CSS selectors that target elements based on multiple conditions such as element types, class names, IDs, attributes, and their relationships in the HTML structure. Examples include descendant selectors, child selectors, adjacent sibling selectors, and attribute selectors.

What is the difference between pseudo-classes and pseudo-elements?

Pseudo-classes are used to define special states or behaviors of elements, such as :hover, :active, or :focus. Pseudo-elements, on the other hand, target specific parts of an element, like the first letter or first line of text, using selectors like ::first-letter and ::first-line.
